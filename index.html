The Rules

1. Any live cell with fewer than two live neighbor's dies, as if by under population.

2. Any live cell with two or three live neighbor's lives on to the next generation

3. Any live cell with more than three live neighbor's dies, as if by overpopulation.

4. Any dead cell with exactly three live neighbor's becomes a live cell, as if by reproduction.

Dead cell = 0 (black) Live cell = 1 (white)
